# v2 тАФ Widget System & Customizable Grid (2026-04-23)

## Design Decisions

1. **Orphan asset cleanup**: No widget references asset тЖТ delete asset + snapshots.
2. **Display name is widget-owned**: `config.label`, not `assets.display_name`.
3. **Default size**: 1x1 compact. **Grid**: 6 cols, no compaction, free placement.
4. **Clean slate**: No v1 data migration.

## Phase 1: Backend тАФ Widget Model + CRUD

- `Widget` model: `id`, `type` (enum: asset/time), `config` (jsonb), `layout_x/y/w/h`.
- 5 endpoints: `GET/POST /widgets`, `PATCH/DELETE /widgets/{id}`, `PUT /widgets/layout`.
- Orphan cleanup on delete. Removed `PATCH /assets/{id}`.
- Migration: `a32d37208a59`.

## Phase 2: Frontend тАФ Types + API + Hook

- Installed `react-grid-layout`. Added `Widget` types, `widgets.ts` API, `useWidgets` hook.

## Phase 3: Frontend тАФ Widget Grid + Edit Mode

- `WidgetGrid` wrapping `ResponsiveGridLayout` (6/4/2/1 cols). `compactType={null}`.
- Edit mode toggle: draggable/resizable + dashed borders + delete buttons.
- Layout debounced 400ms тЖТ `PUT /widgets/layout`.

## Phase 4: Asset Widget тАФ Responsive Price Cards

- Backend: Added `day_high`, `day_low`, `volume` to `PriceSnapshot`. Migration `e55cbd8eb5b9`.
- 4 size variants: 1x1 compact, 2x1 wide, 1x2 tall, 2x2 full.
- Sparkline gradient fix: `useId()` unique IDs per widget.

## Phase 5: Time Widget тАФ Live Clocks

- `TimeWidget`: `Intl.DateTimeFormat` + `setInterval`. Analog/digital mode.
- `AnalogClock`: SVG clock face. 4 responsive variants.

## Phase 6: Add Widget Modal

- Two-step: type picker тЖТ config. Asset uses Yahoo Finance autocomplete. Time uses timezone search.
- Backend: `GET /timezones` endpoint.

## Phase 7: Polish + Cleanup

- Edit widget modal. Dead code removal (~200 lines CSS).
- Version bump: `1.1.0` тЖТ `2.0.0`.
- Fast-polling for new assets (3s until price appears).
- Viewport-fit layout with styled scrollbar.

### State at v2.0.0
- 3 tables: `assets`, `price_snapshots`, `widgets`.
- Widget grid with asset + time widgets, drag/drop, resize, edit mode.
